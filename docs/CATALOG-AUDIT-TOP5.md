# Catalog audit — top 5 states (CA, TX, FL, NY, IL)

Audited for Scholarship One product pass. Deadlines labeled as cycle/soft when not a single fixed calendar day.

| State | Program IDs | Official hub | Notes |
|-------|-------------|--------------|-------|
| California | `cal-grant`, `ca-middle-class` | csac.ca.gov | FAFSA/CADAA cycle; GPA verification for Cal Grant |
| Texas | `texas-grant`, `texas-tfg` | highered.texas.gov | TEXAS Grant + campus aid hub; priority dates vary |
| Florida | `bright-futures-fl`, `florida-fsag` | floridastudentfinancialaidsg.org | Merit (Bright Futures) + need (FSAG) |
| New York | `excelsior-ny`, `ny-tap` | hesc.ny.gov | Excelsior income/residency rules; TAP is primary need grant |
| Illinois | `illinois-map`, `illinois-aim-high` | isac.org | MAP is FAFSA-priority; AIM HIGH is campus-run |

**Rule:** soft/rolling/cycle deadlines show “Confirm official” in the UI. Fixed ISO dates show day countdown.
